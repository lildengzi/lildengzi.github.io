# 寄存器
CPU中存储数据的器件，一个CPU有多个寄存器
AX,BX是两个寄存器的代号

# 汇编组成
1. 汇编指令(机器码助记符)
2. 伪指令(编译器执行)
3. 其他符号(编译器识别)
核心为指令，他决定了汇编的特性

# 存储器
