
## 输入输出
c语言标准输入输出类型
### printf
`printf();`
c语言标准输出
```c
printf("Hello World");
printf("%s", "Hello World");
printf("%d %f %c %c", 10, 10.1, 'A', '\n');
```

### scanf
`scanf();`
C语言标准输入
```c
scanf("%d", &a);
scanf("%f", &b);
```
[[占位符和ASCII码]]

### cout/cin
`cout`
`cin`
c++标准输入输出类型
c++标准输入输出,包含于命名空间std中
```cpp
int a = 0;
cin >> a;
std::cout << a << "Hello World" << std::endl;
//output:0
```
此外，c/c++还有大量其他输入输出方式
