#### 各式各样的 Hello world
```c
#include<stdio.h>

int main()
{
	printf("Hello World");
	return 0;
}
```

```cpp
#include<iostream>
using namespace std;

int main()
{
	cout << "Hello World" << endl;
	return 0;
}
```
​
```java
// Java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}
​```
